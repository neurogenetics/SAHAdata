# SAHA-data
This download-able package will allow instant load-in of Allen Brain Cell Atlas (mouse single cell; Yao et al., 2023 Nature PMID38092916) markers and average expression as calculated by SAHA authors. In the future, SAHAdata will be deprecated as loading data from GitHub is time/resource inefficient. Please look at the associated zenodo of SAHAdata where additional databases will be deposited for further use DOIXXXXXXX.
